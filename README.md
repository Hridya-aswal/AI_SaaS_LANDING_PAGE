# Grocify – Smart Kitchen Inventory Landing Page

## Overview

Grocify is a modern SaaS-style landing page for a smart kitchen inventory platform. The website showcases AI-powered features that help users track groceries, reduce food waste, discover recipes, and manage household inventory efficiently.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern dark theme with green branding
- Interactive navigation bar
- Hero section with call-to-action buttons
- Feature cards with hover animations
- Customer testimonials section
- Pricing plans (Starter, Pro, Family)
- FAQ accordion using JavaScript
- Smooth scrolling navigation
- Scroll reveal animations
- Interactive pricing card highlighting

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Media Queries

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No installation required.
